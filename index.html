Hola, soy Flor Guadalupe Llamas Zamorano

Conoce a Flor

Actualmente tengo 21 años y soy originaria del estado de Aguascalientes, México.

Me encuentro cursando la carrera de Ingenieria en Gestión y Desarrollo de Software Multiplataforma. He realizado varios proyecto de desarrollo de aplicaciones web y móviles, de administración de base de datos, construcción de APIs, desarrollo de videojuegos.

Así mismo, cuento con las siguientes certificaciones: ICP Developer Azle, ICP Motoko Developer, ICP Motoko Certified Developer, Certificación en negociación.

También cuento con las siguientes constancias: Constancia desarrolo Frontend, Constancia de curso de inglés.

Mi parte favorita de la carrera es administrar los diversos tipos de bases de datos existentes, además de contar con experiencia en ello. De igual forma, me gusta mucho dar funcionalidad y diseño a las aplicaciones y páginas web.

Mis proyectos

Estos son algunos proyectos que he desarrollado

Coco wilson

Aplicación de escritorio, desarrollada con C#, para la administración de usuarios, ordenes y pedidos de un retsaurante, cuenta con un menú de comida.

Sitio web publicitario

Sitio web publicitatio para promocionar la carrera de Entornos Virtuales y Negocios Digitales de la UTNA, desarrollado con HTML, JS y CSS.

API REST Restaurante

API REST desarrolada para realizar el ABC de platillos de un restaurante, permite registro de usuarios. Usa XAMPP como SGBD y es desarrolada con JavaScript Y Node.js

API Transacción

API para transacciones de Bitcoin, donde un usuario puede transefir a otro la cantidad deseada. Realizada con el uso de Azle.

API REST Compras

API REST para el manejo de ABC de las compras de una tienda. Usa XAMPP como SGBD y es desarrolada con JavaScript Y Node.js

API REST Polaina's Candys

API REST para la dulceria Polaina's Candys, permite administrar usuarios y sucursales, realizar ventas, así como crear un carrito de productos. Usa XAMPP como SGBD y es desarrolada con JavaScript Y Node.js

API REST Autolavado

API REST para el manejo de empleados de un autolavado, donde un jefe puede registrarlos y relacionarlos a él con el ID del empleado. Ambos pueden realizar un ABC de los servicios ofertados. Usa Mongo DB como SGBD y es desarrolada con JavaScript Y Node.js

API Paqueteria

API encaminada al manejo de paquetes, permitiendo acciones de ABC para paquetes. Dicha API es codificada con Azle.

Casa inteligente

Sitio web para el control de los elementos de una casa inteligente, dicho sitio es desarrollado con PHP, JavaScript, Node.js y utiliza XAMPP como SGBD.

Hidro servicios

Sistema para servicio de pipas de agua, cuenta con registro de usuarios, realizar y ver pedidos, envío de comentarios sobre el servicio para su mejora. Integra Motoko y React.

Agencia de viajes

Proyecto de diseño para una agencia de viajes, centrado en ofrecer una esperiencia de usuario completa, así como ser accesible, usable y funcional.

Rescate en la granja

Videojuego 3D, cuyo objetivo es recoger animales, uir de coyotes y eliminarlos en determinado tiempo, ello mientras recorre laberintos diversos de acuerdo al nivel.

Memorama

Aplicación movil, la cuál es un videojuego de memoria con 3 niveles de dificultad. Permite ingresar un nombre de usuario para guardar los datos de su partida, visualizar puntuaciones e instrucciones. Es desarrollada con HTML, JS, CSS y Apache Cordova es usado para empquetar el proyecto como una apliacioón móvil.

My Pet App

Aplicación móvil para el cuidado de mascotas, donde se permite realizar ABC para las vacunas, enfermedades y citas de la mascota. Así mismo, permite ver y editar el perfil de la mascota. Cuenta con una sección de contactos de emergencia, una con información de la empresa y otra con recomendaciones acerca de clínicas veterinarias en aguascalientes.
